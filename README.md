# Ecommerce-MIS-Analysis

E-commerce Sales & Operations Analysis
📌 Project Overview

This project focuses on analyzing sales, profitability, customer behavior, and delivery performance for an e-commerce business using SQL, Excel, and Power BI.

The objective was to simulate a real-world MIS Executive role by building an end-to-end reporting and analytics solution.

 Tools & Technologies :
SQL (PostgreSQL)
Excel (Data Cleaning & Transformation)
Power BI (Dashboard & Visualization)

Dataset:

The dataset consists of:

Orders data
Customer details
Product information
Shipment and delivery records
 
 Key Analysis Performed:
Revenue and profit analysis
Loss-making product identification
Regional performance analysis
Customer segmentation (New vs Repeat)
Delivery delay and courier performance analysis
 
 Key Insights:
High discounting is the primary driver of loss-making orders
Electronics category generates the highest revenue but lower profit margins
Certain regions contribute high revenue but underperform in profitability
Repeat customers contribute a significant share of total revenue
Delivery delays are higher with specific courier partners
 
Dashboard

The Power BI dashboard includes:

KPI overview (Revenue, Profit, Orders)
Sales trends
Regional performance
Product profitability
Customer segmentation
Delivery performance
🚀 Outcome

This project demonstrates the ability to:

Work with structured datasets
Perform business-oriented analysis
Build dashboards for decision-making
Derive actionable insights
